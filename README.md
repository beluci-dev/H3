## H3js a simple template engine.

You can use the [editor on GitHub](https://github.com/Heronbeluci/H3/edit/master/README.md) to maintain and preview the content for your website in Markdown files.

Whenever you commit to this repository, GitHub Pages will run [Jekyll](https://jekyllrb.com/) to rebuild the pages in your site, from the content in your Markdown files.

### Hello World

```markdown
var HelloWorld = new H3.Block(function(){

	this.elem('h1', {text:'Hello World!'});

}).build().render(document.body);
```

For more details see [GitHub Flavored Markdown](https://guides.github.com/features/mastering-markdown/).

### Jekyll Themes

Your Pages site will use the layout and styles from the Jekyll theme you have selected in your [repository settings](https://github.com/Heronbeluci/H3/settings). The name of this theme is saved in the Jekyll `_config.yml` configuration file.
